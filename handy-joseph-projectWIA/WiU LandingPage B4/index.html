Project Name: WorldSurf

Project Description

WorldSurf.com will be the first web 2.0 surf portals available on the Internet, including mobile devices. The site will serve as an information portal to the public providing the highest visibility of content on various surf destinations around the world. This page will be utilized as a testing page for user interface designs. Those links below will provide data that will explain how a user navigates a user interface.
